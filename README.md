
## nest-test

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## help
```bash
# 创建service文件
$ nest g service <filename> <dirname>

# 创建controller文件
$ nest g controller <filename> <dirname>

# 创建module文件
$ nest g module <filename> <dirname>
```
